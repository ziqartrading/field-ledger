# Static client runtime

Field Ledger v3.1.7 durable offline client with an encrypted dedicated-folder mirror. GitHub Pages publishes from `main /docs`.

Business data, backups, passwords and Apps Script source must never be committed to this public repository.

The visible local folder contains encrypted `.fld` files. Keep `device.json`, the encrypted files, and the recovery password private.

Login-screen hotfix: recovery is available only after sign-in in Admin; sign-in now shows an animated progress indicator and current connection stage.
